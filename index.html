Cisco Network Device Inspection Checklist

# Checklist Item Status Notes
1. Hardware Inspection
1.1 Physical condition (No damage, dust, loose components)
1.2 All LEDs functioning properly
1.3 Cooling system operational (fans, vents)
1.4 Power supply status (Primary and redundant if applicable)
2. System Health
2.1 CPU utilization (< 75% under normal load)
2.2 Memory utilization (< 80% usage)
2.3 Interface errors/drops (check with 'show interface')
3. Security Configuration
3.1 Access control lists (ACLs) properly configured
3.2 SSH enabled (Telnet disabled)
3.3 Strong passwords configured
3.4 Port security enabled on access ports
4. Network Configuration
4.1 VLAN configuration verified
4.2 Spanning Tree Protocol configuration
4.3 Routing protocols configuration (if applicable)
5. Monitoring & Management
5.1 SNMP configuration secure and functional
5.2 Syslog configured and operational
5.3 NTP configuration verified
